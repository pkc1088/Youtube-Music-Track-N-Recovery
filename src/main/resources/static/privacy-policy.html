개인정보처리방침

최종 업데이트: 2025년 06월 04일

1. 수집하는 정보

Google OAuth2 인증을 통해 다음 정보를 수집합니다:

  • Google 계정 기본 정보 (이메일 주소, 고유 식별자)
  • YouTube 데이터 액세스를 위한 액세스 토큰 및 리프레시 토큰
  • 사용자의 YouTube 재생목록 및 동영상 메타데이터

2. 정보 사용 목적

본 서비스는 YouTube 재생목록 항목을 삭제 및 추가하는 기능을 구현하기 위해 Google에서 지정한 민감 권한인 `youtube.force-ssl` 범위를 사용합니다. 해당 권한은 Google OAuth 인증을 통해 사용자로부터 명시적으로 동의를 받아 사용되며, 다른 범위의 권한은 요청하지 않습니다. 수집한 정보는 다음 목적으로만 사용됩니다:

  • 서비스 제공을 위한 사용자 식별
  • YouTube 재생목록 변경 사항 추적
  • 재생목록 내 비정상 동영상 삭제 및 새 영상 추가

3. 데이터 보관 및 삭제

사용자 데이터는 다음 기준에 따라 관리됩니다:

  • 회원 탈퇴 시 모든 개인정보 및 연동 데이터 즉시 삭제
  • 수집된 사용자 데이터는 Google Cloud SQL 인스턴스에 저장
  • 해당 데이터베이스는 암호화 및 접근제어를 통해 보호
  • 서비스 제공 목적 외에는 어떠한 방식으로도 접근 금지
  • 리프레시 토큰은 Google OAuth 철회 API를 통해 즉시 무효화

4. 제3자 공유

본 서비스는 Google 사용자 데이터(YouTube 재생목록, 토큰 등)를 Google API (YouTube Data API v3)에 전송하여 사용자 요청에 따른 재생목록 수정 기능을 수행합니다. 사용자 데이터는 다음 경우를 제외하고 어떠한 외부 제3자(예: 광고업체, 외부 서버 등)와도 공유하지 않습니다:

  • Google YouTube API 와의 정상적인 서비스 운영을 위한 전송
  • 법적 요구사항이 있는 경우

5. 보안 조치

데이터 보호를 위해 다음 조치를 시행합니다:

  • 최소 권한 원칙에 따른 엄격한 접근 제어
  • 정기적인 보안 감사

6. Google 사용자 데이터 처리 흐름 요약

전반적인 서비스 등록 및 사용자 데이터 처리는 다음의 순서를 따릅니다:

  • 사용자는 OAuth 인증을 통해 이메일 및 YouTube 데이터에 대한 접근을 허용합니다.
  • 본 서비스는 해당 정보를 사용하여 재생목록을 분석하고, 비정상 동영상을 탐지합니다.
  • 1일 1회, 감지된 항목에 대해 곡 정보를 유추하고, YouTube API를 통해 대체 동영상을 삽입합니다.
  • 이 과정에서 토큰 및 재생목록 정보는 Google Cloud SQL에 저장됩니다.
  • 사용자가 계정을 삭제하면 모든 정보가 즉시 삭제되며, Google OAuth 토큰도 철회됩니다.
