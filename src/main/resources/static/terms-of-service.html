서비스 이용약관

최종 업데이트: 2025년 06월 04일

1. 서비스 개요

본 서비스(YouTube Playlist Track & Recovery Service)는 사용자의 YouTube 재생목록을 모니터링하고, 비정상 처리된 동영상을 복구하는 기능을 제공합니다.

2. 서비스 이용 조건

  1. 만 14세 이상만 서비스 이용이 가능합니다.
  2. 서비스 이용을 위해 Google 계정 연동이 필수적입니다.
  3. YouTube API 사용 정책을 준수해야 하며, 다음 행위는 금지됩니다:
    • 스팸 또는 대량 자동화 작업
    • 저작권이 있는 콘텐츠의 무단 복제
    • API 호출 횟수 초과를 유발하는 행위

3. 권리 및 의무

사용자의 권리:

  • 언제든지 Google 계정 연동 해지 가능
  • 회원 탈퇴를 통해 모든 데이터 영구 삭제 요청 가능

서비스 제공자의 권리:

  • 이용약관 위반 시 서비스 이용 제한
  • 서비스 운영을 위한 필요 최소한의 데이터 처리

4. 책임 제한

본 서비스는 다음에 대해 책임을 지지 않습니다:

  • YouTube 정책 변경으로 인한 기능 제한
  • 사용자의 과실로 인한 데이터 손실
  • Google 서비스 중단 등 불가항력적 사유로 인한 서비스 장애

5. 약관 변경

약관 변경 시 서비스 내 공지사항 또는 이메일을 통해 고지하며, 변경 후 7일 간 이의 제기가 없을 경우 효력이 발생합니다.

6. 분쟁 해결

서비스 관련 분쟁 발생 시 대한민국 법률에 따라 해결합니다.

위 약관에 동의하지 않을 경우 서비스 이용을 중단하시기 바랍니다.
